# Homepage for YunLeFun

- ☁️ Preview: [yunle.fun](https://yunle.fun)

## Dev

```bash
# install dependencies
pnpm i
# start http://localhost:3000
npm run dev
```

## Deploy

```bash
# generate static files in `.output/public`
npm run generate
```
