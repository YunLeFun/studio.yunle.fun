# Studio for YunLeFun

云乐坊工作室

- ☁️ Preview: [studio.yunle.fun](https://studio.yunle.fun)

## Dev

```bash
# install dependencies
pnpm i
# start http://localhost:3000
npm run dev
```

## Deploy

```bash
# generate static files in `.output/public`
npm run generate
```
